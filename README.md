## 🎬 Introduce
> Solo en casa -> '나 홀로 집에'라는 뜻으로 방송에서 여러 사람들이 함께 하는 게임들을 혼자서도 즐길 수 있는 퀴즈 웹사이트입니다. <br>


## ⚙️ Tools
![Architecture]
> • Database → Oracle 11.2.0.2.0 <br>
> • Web → Html, CSS, Jquery, JavaScript <br>
> • Library → BootStrap <br>
> • Tool → Spring, SqlDeveloper
